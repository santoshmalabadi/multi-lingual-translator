# multi-lingual-translator
AI based Multi - Lingual Content Translator and Generation System

This project is an AI-powered web application that translates text into multiple languages and generates summaries using Natural Language Processing.

Technologies Used:
- Python
- Flask
- HTML/CSS
- NLP
- Translation APIs

Features:
- Multi-language translation
- Text summarization
- Web-based interface

Future Improvements:
- Voice translation
- Document translation
- AI content generation
